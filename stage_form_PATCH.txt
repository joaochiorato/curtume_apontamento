LOCALIZAÇÃO DO AJUSTE NO stage_form.dart
=========================================

Procure pela linha que contém:

    _timeRow(),
    const SizedBox(height: 12),
    _fulaoSelector(),                    ← ESTA LINHA
    const SizedBox(height: 10),

SUBSTITUA POR:

    _timeRow(),
    const SizedBox(height: 12),
    if (widget.stage.hasFulao) ...[     ← ADICIONAR CONDICIONAL
      _fulaoSelector(),
      const SizedBox(height: 10),
    ],

EXPLICAÇÃO:
- O "if" verifica se o estágio tem Fulão
- Os "..." spread operator incluem todos os widgets
- Apenas REMOLHO tem hasFulao = true

RESULTADO:
- REMOLHO: Mostra Fulão e botão Químicos
- Outros estágios: Não mostram esses campos
